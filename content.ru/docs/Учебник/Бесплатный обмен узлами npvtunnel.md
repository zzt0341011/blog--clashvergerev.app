# Как импортировать конфигурацию в NPV Tunnel

1. Скопируйте один из узлов ниже.
2. Откройте приложение **NPV Tunnel**.
3. Перейдите в раздел **Configs**.
4. Нажмите кнопку **+**.
5. Выберите **Import Config from Clipboard**.
6. Вставьте ссылку на конфигурацию.
7. Введите имя и сохраните конфигурацию.

# Импорт конфигурации из буфера обмена

* Скопируйте любой узел из списка выше.
* Откройте **NPV Tunnel**.
* Перейдите в раздел **Configs**.
* Нажмите **+ → Import Config from Clipboard**.
* Вставьте ссылку и сохраните конфигурацию.

Некоторые бесплатные узлы могут перестать работать со временем. Если один сервер недоступен, попробуйте другой.

Обычно нет. Бесплатные узлы могут иметь:

* низкую скорость;
* ограничения по пропускной способности;
* нестабильное соединение.

Для ежедневного использования рекомендуется приобрести платную подписку.
